# தனித்தமிழ்ஞாலக்களஞ்சியம்

[TTAK](https://thanithamizhakarathikalanjiyam.github.io/) நிரல் திட்டங்களில் அடுத்த முனைப்பு. இதில் தனிநபர் உருவாக்கிய தமிழ் அகராதிகள் தொகுக்கும் இடமாகும்.

பல சொற்களஞ்சியங்கள் கொண்ட தேடு பொறி அனைவரது கணிணிகளிலும்/கைப்பேசிகளிலும் இருக்க வேண்டும் என்பது அவா.

## 

மிகு விரைவில் அகி மூர்த்தி அவர்களின், அறிவியல் அகராதி களஞ்சியம் வெளியிடப்படுகிறது.
