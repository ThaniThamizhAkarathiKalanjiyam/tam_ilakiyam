    ‘  ’ ( Cosmos )            .     .        , ,      ‘   ’ ( Charles Darwin )      .

              .       .  24,000   ,       .           ( Venezuela ) ,  ( Colombia ) ,  ( Ecuador ) ,  ( Peru ) ,   ( Mexico ) ,  ( Cuba )    .            ‘   ’  ( Andes mountains ) ,    ‘  ’ ( Mount Chimborazo in Ecuador )     ,  ( Venezuela )  ,  ‘  ’  ( Orinoquia/Orinoco River ) .    ,  1,700    .    ,      .

     ‘  ’  ( Curare plant )           .  ( Malaria )    ( quinine )    ( Cinchona tree bark )     .         ( Pacific water current ) ,       .   ‘   ’     .     ( Electric Eel )    .   ,   ,            .         .

              .           .                .       .  ,        .            ‘   ’ ( Edward Whymper, 1886 )       .          ‘    ’ ( the theory of unity of nature )      .

  ( Isotherms ) :
       ( Isotherm ) .            .            .          .         .    , ,   ,  ,  ( air pressure ) , ,   ( temperature ) ,   ( shape of the land ) ,   ( strength of magnetic field ) ,    ,             .

        ,    ‘  ’ ( Isotherm )  .      ( contour line maps ) .         ( ,  )      .    ,       ( connection with the ‘ habitats ’ and the species living at that place )    .          .     ‘  ’ ( topographic maps )        .        ,        .

‘    ’ ( the unity of nature ) :
_ “ Everything Is Interrelated ” —Alexander von Humboldt_

     ( visual-presentation of scientific data as maps, charts and graphs )      .          .        .     ( ecology )    . , ,     ( intersconnectedness of all nature, intersconnectedness of geography )      ,       .     ( forces make up nature ) .      ( interrelated forces in nature ) .           ( balance of forces in nature )    .   ‘    ’ ( the unity of nature )  .

  :
  , ,      .  , , , , ,        .    ,      ( Isotherm ) ,    ( the unity of nature )    .           .

( 1 )             , ( 2 ) ‘   ’          ‘    ’      , ( 3 )         ,              .

:
 ,        ( role of environment on diversity of life )           ;             .

    ,     .         ,            .         .    “   ” ( autism spectrum disorders – ASD )  ,   “   ”   .

          .        1925   “    ” ( Niels Erik Nørlund )     . “     ” ( Royal Danish Geodetic Institute )      .      ,       ,   .

               .  1928            .  70       .  105  ,      ,   ,        1993     21   .

             .     .           .

          .      ( own axis )  ,        .

        ,       ?

  “   ” ( motion sickness )    .

     ,       . , ,      .  , ,       seasickness, car sickness, airsickness      .        ,          .

 ( nausea ) ,  ( vomiting ) ,  ( dizziness ) ,  ( fainting ) ,  ( headache ) ,  ( fatigue )      .

  2.   :

     , ,       .      ,         .              .
