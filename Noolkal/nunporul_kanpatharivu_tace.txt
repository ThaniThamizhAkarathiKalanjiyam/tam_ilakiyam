 **   ?**  
    ( one of the founders of   **Biogeography**  )      ‘    ’   ( Alexander von Humboldt / Friedrich Wilhelm Heinrich Alexander von Humboldt – 1769-1859 ) .       ,           .              .   ,        .       .

  ,  ‘   ’   ( Thomas Jefferson, the third President of the United States )   “        ”   .     ‘   ’   ( Simón Bolívar, Venezuelan military leader )   “      ”   .          .       ‘   ’            .      ( center of intellectual life in Paris )     .    ,   ,       .

     ‘  ’   ( Cosmos )             .     .        , ,       ‘   ’   ( Charles Darwin )       .

 _ ** “ I am at present fit only to read Humboldt; he like another sun illuminates everything I behold ”  ― Charles Darwin**_

         /     .       ‘   ’   ( Origin of Species )         .           .        .        .

 ** :**  
     1769     14  ,         ,      ,    .       .       ( electrode )             .

      ‘   ’   ( Philosophy of Science )    ,     ,  ,   ,     ,          .  ,         ‘  ’   ( Geology )      /  ( Geologist )   .   ,    , ,   ( Biology, Geology, Botany )      .             .    ,           .

         .        .         .     ‘   ’   ( Aimé Bonpland, French Botanist )    .          .           .       ‘   ’    ‘   ’      .        ,        .           .    ( ? )      ‘   ’     .

 **    ( 1799-1804 ) :**  
              .       .  24,000   ,       .            ( Venezuela ) ,   ( Colombia ) ,   ( Ecuador ) ,   ( Peru )  ,    ( Mexico ) ,  ( Cuba )     .             ‘   ’   ( Andes mountains )  ,     ‘  ’   ( Mount Chimborazo in Ecuador )      ,  ( Venezuela )   ,   ‘  ’    ( Orinoquia/Orinoco River )  .    ,  1,700    .    ,      .

      ‘  ’    ( Curare plant )            .   ( Malaria )      ( quinine )      ( Cinchona tree bark )      .          ( Pacific water current ) ,       .     ‘   ’      .      ( Electric Eel )      .   ,   ,            .         .

              .           .                .       .  ,        .             ‘   ’   ( Edward Whymper, 1886 )        .           ‘    ’   ( the theory of unity of nature )       .

        ( Isotherm )  .            .            .          .         .    , ,   ,  ,   ( air pressure ) , ,    ( temperature ) ,    ( shape of the land ) ,    ( strength of magnetic field ) ,    ,             .

ISOTHERMS

        ,     ‘  ’   ( Isotherm )   .       ( contour line maps )  .          ( ,  )       .    ,        ( connection with the  ‘ habitats ’  and the species living at that place )     .          .      ‘  ’   ( topographic maps )         .        ,        .

 ‘  **   ’   ( the unity of nature ) :**  
 _ “ Everything Is Interrelated ” —Alexander von Humboldt_

      ( visual-presentation of scientific data as maps, charts and graphs )       .          .        .      ( ecology )     . , ,      ( intersconnectedness of all nature, intersconnectedness of geography )       ,       .      ( forces make up nature ) .       ( interrelated forces in nature ) .            ( balance of forces in nature )     .    ‘    ’   ( the unity of nature )   .

 **  :**  
  , ,      .  , , , , ,        .    ,       ( Isotherm ) ,     ( the unity of nature )     .           .

          .        1925    “    ”   ( Niels Erik Nørlund )      .  “     ”   ( Royal Danish Geodetic Institute )       .      ,       ,   .

               .  1928            .  70       .  105  ,      ,   ,        1993     21   .

     , , ,  .                  .        1971    ,       “    ”   ( William Bowie Medal )   .           .     ,   .      ( asteroid 5632 )  .       ,         .

            ,        ,         .

          ,              :

1\.         ?

2\.   ,        ?

3\.   ?  (   ?   ?    ? )      .

     ‘   ’   ( Richter scale )     ,      ‘    ’   ( Moment Magnitude Scale – MMS ) .  ‘ Mw ‘    ‘ M ’    .          . M 3.0               .

          ,             .          ,        ?,    ?            .

,               ,         .

       .

1\.    ( Hydraulic Fracturing ) ,     M 4.6     (    )  .

2\.       ( Oil Production/Extraction ) ,     M 7.0     (  )  .

3\.    ( Wastewater Disposal ) ,     M 5.6     (  )  .

4\.      ( Enhanced Oil Recovery ) ,     M 4.5    .

    ,               .     ‘   ’   ( Co-produced water/wastewater )   .        ( Reused as frac fluid ) ,   ,                 .         .        .

         ,  2014 – 2017           ( Satellite radar interferometry / interferometric synthetic aperture radar or InSAR )  .    4,000      .         .          .           ( ground movement associated with oil activity )    ,   ,         .

;  ( 1 )       ,   ( 2 )         ,  ( 3 )  /  ,  ( 4 )     ,  ( 5 )   ,  ( 6 )            [  ( 1 )  wastewater injection,  ( 2 )  CO2 injection for enhanced oil recovery  ( EOR ) ,  ( 3 )  salt/limestone dissolution,  ( 4 )  freshwater impoundment in abandoned wells,  ( 5 )  sinkhole formation in salt beds, and  ( 6 )  hydrocarbon production ]     .               .

        ,    ,    ...    .     ,   ,               .

        ,    ‘   ’   ( Peltier tiles )   .               ,     .

      ‘   ’   ( Peltier effect )   . ,             ‘   ’  .           .          .         ( ceramic tiles )  .    ( semiconductors )  .        ,   ,             .

         ,       . ,      ,                ,            .        ,           .       ,       ,                 .       ,             .

! [  ]  ( http://siragu.com/wp-content/uploads/2014/05/chernobyl10.jpg ) ா              .    ,   .   ,     .                   .    ,   “  ”   ( The Red Forest )  .

   ‘    ’   ( milliroentgens per hour – mR/h )   .  [ :   ,      .       ''  ( microSieverts per hour )      .          ] .   ,       ா       .

   ( The first zone  )  –   500 mR/h       .    .

   ( The second zone )  –   200–250 mR/h      25–40%  .

   ( The third zone )  –   50–200 mR/h        .       .

    ,                 .      ா        ( University of South Carolina, Biologist Timothy A.   120   . ,         .

    .   ,                  .     ,        .

,  ,           .  ,     ‘   ’   ( Dmitri Mendeleev )   1871         .

    ,          (   )  .     ,                     .     .

                . ,           ,         .        .

           ,         ‘    ’   (    ) ,  ‘   ’   (    )    .          ,         .

 1871       .   1875       ‘      ’   ( Paul Emile Lecoq de Boisbaudran )   ‘  ’   ( gallium )    .     ‘  ’   ( Gaul )       .      .    , ,          .

  ‘   ’         ‘  ’   .    1937,         70  ,   30    .

     , 1907          . , 1955       ( University of California, Berkeley )  ,   101         ( mendelevium )   .      .    800   . ,        15   ,    .

        .      .     .         .          .

      .    ,    ,      .        ,    .   ( biology ) ,   ( ecology ) ,    ( evolution )        ,            .      ,      .         .

    ‘   ’   ( Charles Darwin ) ,     ‘   ’   ( Ted Case ) ,  ‘    ’   ( John Tyler Bonner )         .                         .

       .       ( buoyancy )      .          .       ,   ,          .      ,        .

   ,       .      .        .    ,           .        15     .       5  .

             .       (  ) ,               (  )           .       .          .

      ,       33      .         .     ,              .

 21, 1969      ‘   ’   ( Neil Armstrong )     ,   .

 “        
      
       ”   
         .

 ,    ,  
 “   ,  
   ? ” 

           .

             .     .           .

          .       ( own axis )   ,        .

        ,       ?

      ‘   ’   ( Tidal locking or captured rotation )     .              .    .        .            .    .     .      ,             .

 ,              .     ‘  ’    ‘  ’    ( Broca ’ s area and Wernicke ’ s area  of the brain )   .

 **1\.   :**

     ,       ,        ,         .     ,           .

       ( Rex Jung and Colleagues )                 ( first approximation )   .    ,        ,     ,     ,     ,       ,                .                            .

,               ,      .

              .              .             .                    .

     ;  ,  ,     ,   ,   ,             .     ,                 .

## 35.    ?

! [  ]  ( https://c1.staticflickr.com/1/150/345589902_34de3e6d49_z.jpg?zz=1 ) 

  

   “   ”   ( motion sickness )     .

     ,       . , ,      .  , ,       seasickness, car sickness, airsickness      .        ,          .

 ‘ Motion sickness ’    ‘ Travel sickness ’       ‘  ’   ( kinetosis )    .               .

 **:**  
  ( nausea ) ,   ( vomiting ) ,   ( dizziness ) ,   ( fainting )  ,   ( headache ) ,   ( fatigue )       .

 **  :**  
  ;          .     ,      ,     .    , ,      ,    ,     .

        ,      ‘   ’   ( vestibular system )   ,  ,        ,       .              ,          .      .  ,      ( Motion sickness caused by motion that is felt but not seen ) .       .      .          ,          ,          ( i.e., Motion sickness caused by motion that is seen but not felt ) .

 **   :**  
         .       .       .    ,      ,          .  ,        , ,       . ,          .      .      ?      ?       ?       .

 ,              .                      .

 **  :**  
     ,         .  ,      ,            .

      ,     ‘   ’     .

   ;          ,         ,      .             .  ,               .           ,          .

    ,         .          ,           .    ,  ‘     ’      .        ..  ( mtDNA )  .     .     ..         ,    .. ,  ..          .  
 [   ( mitochondria )    “  ”    “  ”   ,        .    ‘     ’     ( Deoxyribonucleic acid – DNA )  ..  .             ,   . ] 

 **  :**  
   ( evolution )  ,   ( fossil )         ‘   ’   ( Homo sapiens )        ‘  ’   ( Ethiopia )    200,000    . ,      ,  60,000   70,000         .                .    10,000     .     ,    ( ice age )       ,    ‘    ’   (  ‘ Out-of-Africa ’  theory )     .    ‘  ’     ( Late Pleistocene Era ) .     ...  ( mtDNA )  ,     ( Maternal Ancestry )    .

 ** ...  ( mtDNA )   ?**  
   ,    ( metabolism )     . ,           ... .  ...    ‘  ’   ( nucleus )   ,       ( cytoplasm )      .   ‘  ... ’   ( mtDNA )    .

  ...    ‘   ’   ( genetic mutation )    ,          .         ...   ,    “  ...  ”  ( mitochondrial DNA haplogroup )    .         ‘  / ’   ( phylogeny )   .   ...       ( matrilineal inheritance )   ,            .

         .     ‘   ’   ( Mitochondrial Eve )   .         ( most recent common ancestor – MRCA ) .    Y-   ( chromosome )  ...    ‘    ’   ( patrilineal line studied by Y – DNA )   .      ‘    ’   (  Y-chromosomal Adam )   .          ( common ancestor )  .

   ,   A  Z  ,   ...    .     ,  ,      .       A – Z   .     L.       L0, L1, L2, L3, M, N, R         .           .    ( macro-haplogroup )     .      ‘   ’   ( sub haplogroup )    .

! [ stem cells ]  ( http://www.vallamai.com/wp-content/uploads/2013/09/stem-cells.gif ) 

     ( leukemia, blood cancer )      .        ( bone marrow )  .        .                 .        ( stem cells transplant ) .        .

        . :  
1\.     ( Adult/Tissue specific stem cells )   
2\.     ( Embryonic stem cells/ Pluripotent stem cells )   
3\.     ( Induced Pluripotent stem cells ) 

1.   :  
         .               .

2.   :  
    .     ( fertility clinic )         .      ,                        .         ( pluripotent stem cells /iPS cells ) .

3.   :  
 , ,       .      ,         .              .

,       ,          . ,       ,                   .

    : 1.    , 2.      , 3.                 .

     (  11, 2013 ) ,    ,              .    ,            .    ,    ,          .

        ,     , ,    .

           .         , .   “  ”  ( nociceptors )   .

      , , , ,        .             .          .

,        ,        .          .             ( arachidonic acid )         .       .

     .      .          .     .  “  ”    “   ”      .

  

 **  ?  **

           .    “ -1   – 2   ”   ( COX-1 and COX-2 enzymes )       “  ”   ( prostaglandins-H2 )    .            , ,        .  

      ( active site )      .     .  -1   – 2    ,        .  ,         .

   ( Neuropathic pain )       .         .

  

 **   ?**

            ,    .    -1   – 2            .

           .         ,     .

   , -1   – 2               .   -1   – 2       .

            ,                 .

        .     ,   ‘   ’   ( Nature Communications, Published: 27 July 2018 )   .   ‘  ’   ( hemihelix )       . , , , ,            ,      .
